# Invoice Project

A robust, full-stack solution for invoice management, built following modern web development practices and system optimization principles.

## Technologies Used

- **Frontend:** React, JavaScript, HTML, CSS
- **Backend:** Spring Boot (MVC, Security, JPA), JWT
- **Database & Caching:** MySQL, Redis

## Features

- **Invoice CRUD:** CRUD invoices/customers/products via RESTful APIs powered by Spring Boot.
- **Authentication & Authorization:** Stateless, secure access control using JWT and Spring Security, with role-based permissions.
